# Homebrew Tap for GAM

This is a Homebrew tap for [GAM](https://github.com/GAM-team/GAM), a command line tool for Google Workspace management.

## Installation

Replace `<user>` with the GitHub username or organization where this repository is hosted.

```bash
brew tap <user>/gam
brew install gam7
```
